# SPEC驱动开发模式
**严格按照SPEC驱动的开发模式**
根据相关skill,遵循Requirements → Design--Task-Develop 的流程
过程中将所有的文档写入doc目录:
Requirements.md:需求文件。根据用户的需求,创建规范的需求文件 
Design.md:设计文件。根据需求,精准拆分该项目的设计方案落地文档
Task.md:根据需求
设计等文档,明确你应该如何开始本项目。并将你的任务落地成该文档
